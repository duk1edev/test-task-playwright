# Events Widget -- Playwright Automated Tests

Автоматизированные тесты для страницы:\
https://dev.3snet.info/eventswidget/

Проект реализован с использованием **Playwright + TypeScript**.

------------------------------------------------------------------------

## 📦 Технологии

-   Playwright\
-   TypeScript\
-   Page Object Pattern\
-   Step-based test architecture\
-   HTML Reporter

------------------------------------------------------------------------

## 🚀 Быстрый запуск

### 1) Клонировать репозиторий

``` bash
git clone <your-public-repo-url>
cd eventswidget-playwright
```

### 2) Установить зависимости

``` bash
npm install
```

### 3) Установить браузеры Playwright

``` bash
npx playwright install
```

### 4) Запуск тестов

``` bash
npm test
```

### Запуск в headed-режиме

``` bash
npx playwright test --headed
```

### Открыть HTML-отчёт

``` bash
npx playwright show-report
```

## 🔹 Tests

Содержат:

-   последовательность шагов\

## 🧪 Реализованные тесты

### 1️⃣ Проверка базовой загрузки страницы

Проверяется:

-   отображение блока **"Шаг 1"**\
-   наличие кнопки **"Сгенерировать превью"**\
-   наличие кнопки **"Скопировать код"**

------------------------------------------------------------------------

### 2️⃣ Позитивный сценарий генерации превью

Сценарий:

1.  Открыть страницу\
2.  Открыть выбор тематики\
3.  Выбрать **"Affiliate"**\
4.  Нажать **"Сгенерировать превью"**\
5.  Дождаться состояния **"Загрузка..."**\
6.  Дождаться появления `iframe`\
7.  Проверить, что в `iframe` отрендерены события:
    -   `div.event-group-sport`\
    -   `tr.event-activity-item`

Проверяется корректность:

-   асинхронной генерации\
-   обновления `iframe`\
-   рендера данных

------------------------------------------------------------------------

### 3️⃣ NEGATIVE: смена языка не переводит пункты меню (Known Issue)

Сценарий:

1.  Проверить, что меню содержит кириллицу\
2.  Переключить язык на **EN**\
3.  Ожидать, что пункты меню будут без кириллицы

Фактическое поведение:

-   URL меняется на `/en/eventswidget/`\
-   индикатор языка меняется на **EN**\
-   пункты меню остаются на русском языке

Тест помечен как expected failure:

``` ts
test.fail(true, 'Known issue: menu items remain in Russian after language switch');
```

Это позволяет:

-   зафиксировать дефект\
-   не ломать общий прогон тестов

------------------------------------------------------------------------

## ⚠ Обнаруженные особенности страницы

-   Выпадающий список языка имеет несколько копий (desktop/mobile), что
    требует точного селектора.\
-   Смена языка инициирует навигацию, но элементы меню не переводятся.\
-   Генерация `iframe` требует явного ожидания состояния
    **"Загрузка..."**.\
-   `iframe` загружается асинхронно --- требуется ожидание контента
    внутри `frame`.

## 📊 Формат отчёта

Используется Playwright HTML Reporter.

После запуска тестов доступен отчёт:

``` bash
npx playwright show-report
```

Отчёт содержит:

-   статус тестов\
-   шаги (`test.step`)\
-   скриншоты при падении\
-   trace при retry
